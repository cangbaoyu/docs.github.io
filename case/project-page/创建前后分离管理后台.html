16-创建前后分离管理后台

2025年8月5日大约 2 分钟SpringCloud Alibaba For Page项目
此页内容
  • 概述
  • 克隆模板到本地
  • 创建项目管理后台
  • 构建并运行

概述

利用 VUE ELEMENT ADMIN 来快速搭建属于自己的产品后台 备注：使用该后台模板开发需要掌握 NodeJS、ES2015+、vue、vuex、vue-router 、axios 和 element-ui 等前端技术栈相关知识

克隆模板到本地

  • 克隆 vue-element-admin 完整模板到本地，主要作用是方便我们直接拿组件到项目中使用
git clone https://github.com/PanJiaChen/vue-element-admin.git
  • 克隆 vue-admin-template 基础模板到本地，主要作用是创建一个最简单的项目后台，再根据需求慢慢完善功能
git clone https://github.com/PanJiaChen/vue-admin-template.git
