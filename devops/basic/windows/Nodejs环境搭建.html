使用云效制品仓库

NPM 配置

通过本指南快速设置仓库凭证，并完成制品文件的推送。

  • 步骤一：设置仓库凭证

设置云效仓库 npm-registry 为 npm 默认仓库，云效 npm-registry 会默认将淘宝镜像（cnpm:https://registry.npmmirror.com）设置为代理仓库。

npm
npm config set registry https://packages.aliyun.com/5f5ad5ce5fd102f22f6bdef6/npm/npm-registry/
pnpm
pnpm config set registry https://packages.aliyun.com/5f5ad5ce5fd102f22f6bdef6/npm/npm-registry/
yarn
yarn config set registry https://packages.aliyun.com/5f5ad5ce5fd102f22f6bdef6/npm/npm-registry/

登陆 npm 仓库并设置凭证 输入用户名、密码

npm login
  • 步骤二：推送 进入你的工程目录，设置好您的package.json，执行以下指令完成推送或者拉取。
推送
npm publish
拉取
npm install [package]

node.js 环境配置

安装的全局模块所在的路径，以及缓存cache的路径(自定义文件夹)

# 设置看全局模块命令
npm config set prefix "D:\Develop\Nodejs\Resources\node_global"

# 设置缓存cache命令
npm config set cache "D:\Develop\Nodejs\Resources\node_cache"

# 配置 pnpm 存储路径
pnpm config set store-dir "D:\Develop\Nodejs\Resources\.pnpm-store"

# 配置 pnpm 缓存路径
pnpm config set cache "D:\Develop\Nodejs\Resources\pnpm_cache"

# 配置 pnpm 全局模块路径
pnpm config set prefix "D:\Develop\Nodejs\Resources\pnpm_global"
# 查看全局模块命令
npm config get prefix

# 查看缓存cache命令
npm config get cache

# 查看 pnpm 源
pnpm config get registry

# 查看 pnpm 缓存路径
pnpm config get cache

# 查看 pnpm 全局模块路径
pnpm config get prefix

# 查看 pnpm 存储路径
pnpm config get store-dir

最后在系统环境变量 PATH 中增加一个 D:\Develop\Nodejs\Resources\node_global
